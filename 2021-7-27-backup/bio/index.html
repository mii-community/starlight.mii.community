Biography

12月 東京都在住の一般人です。
趣味としてプログラミングを始めてそろそろ2年ぐらい経つと思います。

とりあえずまとめてみました。同じ趣味の人がいたら嬉しいな。

趣味

  • プログラミング

    JavaScript や Ruby を主に書いてます。
    もともと「Hot Soup Processor」という言語でプログラミングをしており、
    そのうち Discord の BOT を開発したい、という意欲が湧き Ruby や JS の学習をはじめました。
    最近は HTML/CSS にも触れています。

  • ゲーム

    PC で「Apex Legends」「Rainbow Six Siege」「VRChat」や、 Switch で「ぷよテト2」なんかをやったりしてます。
    音ゲーも好きです。iPad で「プロセカ」などをしてます。技量は初心者のほうが上手いぐらい。
    テトリスが最も好きです。「TETR.IO」のレート 22000 ぐらいでうろちょろしてます。マッチしたらよろしくね。

できること

そこまでありませんが一応。

  • 言語

    一応日本人なので人並みには日本語使えます。多分。
    多少英語も読めます。話せませんし書けませんが。

  • 書けるプログラミング言語

    JavaScript 、 Python がちょっとだけ書けます。あと HTML 。
    そのうち TETR.IO か ぷよテト2 用の AI も書けるようになりたいので、 C++ (C#) を学習しようか迷ってます。

  • 好きなゲーム

    シージ や Apex などのシューター系ゲームが好きです。
    また、 TETR.IO みたいなテトリスも好きです。てかこっちのほうが大好き。
